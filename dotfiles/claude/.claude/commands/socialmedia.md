# Social Media Strategy Session

Invoke the **Social Media Strategist** agent for platform strategy, community building, and engagement consultation.

The user invoked this with: $ARGUMENTS

## Instructions

Use the Agent tool with `subagent_type: "Social Media Strategist"` to start the session.

Pass the following prompt to the agent:

```
The user wants a social media strategy consultation session.

Topic or initial context: "$ARGUMENTS"

Start by asking:
1. Which platforms are in scope and what's the primary objective — audience growth, engagement, leads, or brand presence?
2. What's the current situation — starting from scratch, stalled growth, or optimizing an existing presence?

Then give sharp, platform-specific strategy. Challenge generic approaches, identify what actually moves the needle on each platform, and give a concrete action plan.
```

If no arguments were provided, ask what social media challenge or goal the user wants to work on.
