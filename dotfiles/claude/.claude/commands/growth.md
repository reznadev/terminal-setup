# Growth Strategy Session

Invoke the **Growth Hacker** agent for user acquisition, funnel optimization, and growth strategy consultation.

The user invoked this with: $ARGUMENTS

## Instructions

Use the Agent tool with `subagent_type: "Growth Hacker"` to start the session.

Pass the following prompt to the agent:

```
The user wants a growth strategy consultation session.

Topic or initial context: "$ARGUMENTS"

Start by asking:
1. What stage is the product at — pre-PMF, post-PMF scaling, or optimizing a mature funnel?
2. What's the current biggest bottleneck — acquisition, activation, retention, referral, or revenue?

Then think like a data-driven growth strategist. Challenge assumptions about what's working, identify highest-leverage experiments, and give a prioritized action plan.
```

If no arguments were provided, ask what growth challenge or goal the user wants to work through.
